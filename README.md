## 소개

> **분석하고 주도하는 것을 좋아합니다.**
> 문제의 본질을 파악하고, 해결 방안을 직접 찾아 실행합니다. — 혼자서도 0 → 100, 팀과 함께라면 더 멀리.

| | |
| :-- | :-- |
| **서비스 기획 · 운영** | 사이드 프로젝트 매칭 서비스 [`waggle.lol`](https://waggle.lol) 기획·운영 |
| **데이터로 판단** | 지표를 분석해 문제를 정의하고 개선점을 찾음 |
| **잇는 빌더** | 기획·디자인·개발을 연결하고, 협업 도구를 직접 제작 |

#### 일하는 방식

- 막히면 혼자 안고 가지 않고, **빠르게 공유하고 정렬**합니다
- 만든 도구는 **문서·온보딩까지** 챙겨 팀이 바로 쓸 수 있게 합니다
- 기획서든 데이터든, **함께 일하는 사람의 언어**로 이야기합니다
